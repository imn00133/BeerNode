# BeerNode commit convention
## convention 생성일
- 21.02.02.
## convention 규칙
- issue 번호
- 상태
  - Created: 파일/앱 생성
  - Added: 완성되지 않은 일부 내용 추가
  - Fixed: 버그 수정
  - Failed: Fixed, Added 대신 사용, 고치거나 추가하려고 했으나 실패
  - Improved: 기능 개선(디자인 개선 등)
  - Completed: 기능 완성
- 제목
  - 추상적이지 않게 적을 것 
  - Ex) app 개선 x -> scrap app 디자인 개선 o
## convention 예시
- Ex) BeerNode-28 Added: 간단한 제목(72자 내외)
